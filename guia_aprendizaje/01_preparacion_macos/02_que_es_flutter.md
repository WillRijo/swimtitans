# Que es Flutter

**Nivel:** Aprender

## Objetivo

Entender que Flutter es una herramienta para crear apps con pantallas, botones, imagenes y movimiento.

## Herramientas que vamos a usar

- Flutter
- VS Code
- Un proyecto de prueba

## Idea principal

Flutter es una herramienta para crear apps.

Con Flutter puedes crear apps para:

- celulares
- tablets
- web
- computadoras

Una cosa importante de Flutter es que casi todo lo que ves en la pantalla se construye con **widgets**.

Un widget puede ser:

- un texto
- un boton
- una imagen
- una columna
- una fila
- una pantalla completa

En Swim Titans usaremos Flutter para crear:

- la pantalla de inicio
- los botones de nado
- la piscina
- el nadador
- la barra de progreso

## Ejemplo mental

Imagina que una app es como construir con bloques.

Cada bloque es un widget.

Juntas varios widgets pequenos y formas una pantalla completa.

## Lo que haremos despues

Despues aprenderemos que es Dart, que es el lenguaje que usamos para escribir codigo en Flutter.
