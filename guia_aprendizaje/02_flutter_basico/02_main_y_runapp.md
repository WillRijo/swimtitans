# main y runApp

**Nivel:** Aprender

## Objetivo corto

Entender donde empieza una app de Flutter.

## Archivos o herramientas que se usaran

- `lib/main.dart`
- Funcion `main`
- Funcion `runApp`

## Resumen de lo que se explicara despues

Se explicara que `main` es el punto de inicio y que `runApp` le dice a Flutter cual widget debe mostrar primero.

## Actividad futura breve

Encontrar `main` y `runApp` dentro de `lib/main.dart`.
