# Instalar Flutter con VS Code

**Nivel:** Practicar

## Objetivo corto

Instalar Flutter usando VS Code para poder crear proyectos nuevos.

## Archivos o herramientas que se usaran

- VS Code
- Extension de Flutter
- Extension de Dart
- Flutter SDK

## Resumen de lo que se explicara despues

Se explicara como instalar las extensiones de Flutter y Dart, y como VS Code puede ayudar a descargar o encontrar Flutter.

## Actividad futura breve

Abrir la lista de extensiones de VS Code y buscar Flutter.
