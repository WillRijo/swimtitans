# Crear estado del juego

**Nivel:** Aprender

## Objetivo corto

Entender que datos cambian mientras el juego esta funcionando.

## Archivos o herramientas que se usaran

- Variables
- `StatefulWidget`
- `setState`

## Resumen de lo que se explicara despues

Se explicara que el juego necesita recordar cosas como posicion del nadador, tiempo, carril y si la carrera termino.

## Actividad futura breve

Hacer una lista de datos que cambian durante una carrera.
