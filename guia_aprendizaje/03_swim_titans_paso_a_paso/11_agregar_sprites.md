# Agregar sprites

**Nivel:** Practicar

## Objetivo corto

Usar imagenes para que el nadador se vea mejor.

## Archivos o herramientas que se usaran

- Carpeta `assets`
- Imagenes o sprites
- Archivo `pubspec.yaml`

## Resumen de lo que se explicara despues

Se explicara que un sprite es una imagen usada en un juego. Tambien se mostrara como agregar imagenes al proyecto y decirle a Flutter que las use.

## Actividad futura breve

Preparar una carpeta `assets` y agregar una imagen de prueba.
