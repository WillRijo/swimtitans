# Variables

**Nivel:** Aprender

## Objetivo

Entender que una variable guarda un dato para usarlo despues.

## Herramientas que vamos a usar

- Dart
- `String`
- `int`
- `double`
- `bool`

## Idea principal

- Una variable es como una cajita con nombre.
- Dentro de la cajita guardamos un dato.
- Ese dato puede ser texto, numero o verdadero/falso.
- Usamos variables para recordar cosas mientras la app esta funcionando.

## Codigo importante

Ejemplos cortos:

```dart
String nombreJuego = 'Swim Titans';
int distancia = 50;
double progresoActual = 0;
bool practicaTerminada = false;
```

## Explicacion paso a paso

`String` guarda texto.

`int` guarda numeros enteros.

`double` guarda numeros con decimales.

`bool` guarda `true` o `false`.

`nombreJuego` guarda el nombre del juego.

`distancia` guarda cuantos metros se van a nadar.

`progresoActual` guarda cuanto ha avanzado el nadador.

`practicaTerminada` dice si la practica termino o no.

## Conectar con Swim Titans

Swim Titans usara variables como:

- `tipoNadoSeleccionado`
- `distanciaSeleccionada`
- `progresoActual`
- `practicaTerminada`
- `elJugadorEstaNadando`

## Lo que haremos despues

Despues aprenderemos estado y `setState`, que sirven para cambiar datos y actualizar la pantalla.

## Actividad

Crea estos ejemplos en una hoja o en un archivo de prueba:

```dart
String nombreNadador = 'Alex';
int distanciaElegida = 50;
double velocidad = 3.5;
bool jugadorListo = true;
```

Pregunta:

Cual de esas variables cambiaria durante el juego?
