# Variables

**Nivel:** Aprender

## Objetivo corto

Entender que una variable guarda un dato para usarlo despues.

## Archivos o herramientas que se usaran

- Dart
- `String`
- `int`
- `bool`

## Resumen de lo que se explicara despues

Se explicara que una variable puede guardar texto, numeros o respuestas de verdadero y falso. Tambien se mostrara como esos datos ayudan a una app.
