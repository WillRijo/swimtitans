# Que es Flutter

**Nivel:** Aprender

## Objetivo corto

Entender que Flutter es una herramienta para crear apps con pantallas bonitas.

## Archivos o herramientas que se usaran

- Flutter
- VS Code
- Un proyecto de prueba

## Resumen de lo que se explicara despues

Se explicara que Flutter ayuda a crear apps para celular, web y computadora usando el mismo lenguaje. Tambien se explicara que en Flutter casi todo lo que se ve en pantalla se llama widget.

## Actividad futura breve

Buscar en una app conocida tres cosas que podrian ser widgets, como botones, textos o imagenes.
