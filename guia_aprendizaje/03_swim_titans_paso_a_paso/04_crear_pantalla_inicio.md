# Crear pantalla inicio

**Nivel:** Practicar

## Objetivo corto

Crear la primera pantalla real de Swim Titans.

## Archivos o herramientas que se usaran

- Archivo de pantalla en `lib`
- `Scaffold`
- `Column`
- `Text`

## Resumen de lo que se explicara despues

Se explicara como crear una pantalla de inicio con titulo, textos simples y espacio para botones. Esta pantalla sera la entrada al juego.

## Actividad futura breve

Crear una pantalla con el titulo "Swim Titans".
