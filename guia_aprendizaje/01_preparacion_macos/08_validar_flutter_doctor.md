# Validar Flutter doctor

**Nivel:** Practicar

## Objetivo corto

Usar `flutter doctor` para revisar si Flutter esta listo.

## Archivos o herramientas que se usaran

- Terminal
- Flutter
- Comando `flutter doctor`

## Resumen de lo que se explicara despues

Se explicara que `flutter doctor` es como una revision medica para Flutter. Muestra que esta bien, que falta y que se debe arreglar.

## Actividad futura breve

Ejecutar `flutter doctor` y anotar los mensajes que no entiendas para revisarlos con calma.
