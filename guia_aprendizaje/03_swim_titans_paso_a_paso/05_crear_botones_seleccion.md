# Crear botones seleccion

**Nivel:** Practicar

## Objetivo corto

Agregar botones para escoger opciones antes de jugar.

## Archivos o herramientas que se usaran

- Pantalla de inicio
- `ElevatedButton`
- Variables de seleccion

## Resumen de lo que se explicara despues

Se explicara como crear botones para elegir opciones como nadador o estilo. Tambien se vera como guardar la opcion elegida.

## Actividad futura breve

Crear dos botones y hacer que uno quede marcado como seleccionado.
