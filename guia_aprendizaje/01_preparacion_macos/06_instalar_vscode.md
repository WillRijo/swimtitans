## Objetivo corto

Instalar VS Code y entender para que sirve al crear apps.

## Archivos o herramientas que se usaran

- macOS

Se explicara que VS Code es el lugar donde escribiremos el codigo. Tambien se mostrara donde estan los archivos, el editor, las extensiones y la terminal integrada.
