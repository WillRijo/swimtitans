# Crear modelos

**Nivel:** Aprender

## Objetivo corto

Entender que los modelos guardan informacion importante del juego.

## Archivos o herramientas que se usaran

- Dart
- Archivos dentro de `lib`
- Clases para nadador, estilo o juego

## Resumen de lo que se explicara despues

Se explicara que un modelo es como una ficha con datos. Swim Titans puede tener datos como nombre del nadador, estilo de nado, carril y velocidad.

## Actividad futura breve

Escribir en una lista que datos deberia tener un nadador.
