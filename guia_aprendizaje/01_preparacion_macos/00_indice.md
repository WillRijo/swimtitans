# Preparacion en macOS

**Nivel:** Aprender y Practicar

## Objetivo

Saber que aprenderas antes de escribir mucho codigo y dejar macOS listo para crear apps con Flutter.

## Herramientas que vamos a usar

- macOS
- Terminal
- VS Code
- Flutter
- Dart

## Lo que haremos en esta seccion

Primero veremos que vamos a crear. Luego aprenderemos que son Flutter, Dart y programar.

Despues prepararemos macOS, instalaremos VS Code, instalaremos Flutter y crearemos el primer proyecto vacio.

## Primera actividad

Revisar una por una las lecciones de esta seccion y confirmar que la computadora queda lista para crear una app Flutter.
