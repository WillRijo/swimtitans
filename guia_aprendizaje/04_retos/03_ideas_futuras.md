# Ideas futuras

**Nivel:** Aprender

## Objetivo

Imaginar como podria crecer Swim Titans despues de la primera version.

## Herramientas que vamos a usar

- Imaginacion
- Proyecto nuevo de Swim Titans
- Flutter
- Lista de ideas

## Idea principal

Una app puede crecer poco a poco.

No hay que hacer todas las ideas de una vez.

Primero se termina una version simple y luego se elige una mejora.

## Idea 1: Sonidos

Agregar sonidos cuando:

- empieza la practica
- el nadador avanza
- llega a la meta

## Idea 2: Mas personajes

Agregar mas nadadores para elegir.

Cada nadador podria tener:

- nombre
- color de gorro
- sprite diferente

## Idea 3: Menu de pausa

Agregar un boton para pausar el juego.

Opciones:

- continuar
- reiniciar
- volver al inicio

## Idea 4: Records locales

Guardar el mejor resultado en la misma computadora.

Por ahora solo pensarlo como idea futura.

## Idea 5: Mas piscinas

Crear piscinas con estilos diferentes:

- piscina azul
- piscina de competencia
- piscina con carriles de colores

## Idea 6: Trofeos

Mostrar mensajes especiales:

- Primera meta
- Completaste 100 m
- Completaste 400 m
- Practicaste todos los estilos

## Idea 7: Animaciones de celebracion

Cuando llega a la meta:

- mostrar confeti
- mostrar una estrella
- mover el texto de felicitacion

## Idea 8: Competencia

Hacer que el modo Competencia funcione en una version futura.

Podria incluir:

- otro nadador
- tiempo limite
- cuenta regresiva

## Idea 9: Lecciones dentro del juego

Agregar una pantalla que explique:

- como se juega Libre
- como se juega Pecho
- como se juega Dorso
- como se juega Mariposa

## Idea 10: Crear tus propios sprites

Dibujar nuevos sprites y reemplazar los existentes.

## Consejo

Una buena idea se puede escribir primero asi:

```text
Nombre de la idea:
Que cambia en el juego:
Por que seria divertido:
Que archivos creo que tocaria:
```
