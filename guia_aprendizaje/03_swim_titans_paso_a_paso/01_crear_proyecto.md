# Crear proyecto

**Nivel:** Practicar

## Objetivo

Crear el proyecto nuevo donde construiremos Swim Titans desde cero.

## Herramientas que vamos a usar

- VS Code
- Flutter
- Terminal integrada
- Carpeta nueva del proyecto

## Idea principal

Ahora vamos a crear una app nueva.

El proyecto terminado `swimtitans_ejemplo` solo sera una referencia.

No vamos a editar el proyecto de referencia.

La nueva app puede llamarse:

```text
swim_titans
```

O, si prefieres algo de practica:

```text
swim_titans_aprendizaje
```

## Paso 1: Elegir donde guardar el proyecto

Se recomienda una carpeta facil de encontrar, por ejemplo:

```text
~/developer
```

O:

```text
~/developer/swimtitans
```

El nombre exacto puede cambiar segun la computadora.

## Paso 2: Crear el proyecto desde VS Code

1. Abre VS Code.
2. Presiona `Command + Shift + P`.
3. Escribe `Flutter: New Project`.
4. Elige **Application**.
5. Elige la carpeta donde se guardara.
6. Escribe el nombre del proyecto:

```text
swim_titans
```

7. Espera a que VS Code cree los archivos.

## Paso 3: Abrir el proyecto

VS Code debe abrir la nueva carpeta del proyecto.

Si no la abre, usa:

```text
File > Open Folder
```

Luego elige la carpeta del proyecto nuevo.

## Paso 4: Ejecutar la app inicial

Puedes usar una de estas opciones.

**Opcion A**

Presiona `F5`.

Elige Chrome si pregunta.

**Opcion B**

Abre la terminal integrada y ejecuta:

```bash
flutter run
```

## Paso 5: Confirmar que funciona

Si aparece la app inicial del contador, el proyecto esta funcionando.

## Si algo sale mal

- Si VS Code no encuentra Flutter, vuelve a Flutter Doctor.
- Si no aparece dispositivo, prueba Chrome.
- Si aparece error, copia el mensaje.
- No borres el proyecto sin revisar el error.

## Lo que haremos despues

En la siguiente leccion limpiaremos la app inicial y crearemos la estructura basica:

- `main.dart`
- `aplicacion.dart`
- primera pantalla

## Actividad

1. Crea el proyecto nuevo.
2. Ejecutalo una vez sin cambiar nada.
3. Escribe en una nota:

```text
Ya puedo crear y correr una app Flutter.
```
