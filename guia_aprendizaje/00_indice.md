# Guia de aprendizaje de Swim Titans

**Nivel:** Aprender

## Objetivo

Entender como usar esta guia para aprender Flutter, Dart y programacion mientras creas una app nueva llamada Swim Titans paso a paso.

## Herramientas que vamos a usar

- Esta carpeta: `guia_aprendizaje`
- Proyecto terminado de referencia: `swimtitans_ejemplo`
- macOS
- VS Code
- Flutter
- Dart

## Que es esta guia

Esta guia es un camino para aprender desde cero. No es solo una explicacion del proyecto ya terminado.

La idea es que primero entiendas cosas pequenas, luego practiques con Flutter, y despues construyas Swim Titans en una app nueva.

## Como usarla

1. Lee las lecciones en orden.
2. Cuando una leccion diga **Aprender**, solo necesitas entender la idea.
3. Cuando una leccion diga **Practicar**, vas a escribir codigo o usar una herramienta.
4. Cuando una leccion diga **Copiar y usar**, puedes copiar una parte dificil primero y entenderla mas adelante.

## Tipos de leccion

- **Aprender:** sirve para entender una idea nueva con palabras simples.
- **Practicar:** sirve para hacer algo con tus manos, como crear archivos, correr la app o cambiar codigo.
- **Copiar y usar:** sirve para usar codigo mas dificil sin tener que entenderlo todo el primer dia.

## Proyecto de referencia

El proyecto terminado esta en `swimtitans_ejemplo`.

Ese proyecto sirve como referencia para ver como queda la app al final. No tienes que memorizarlo ni copiarlo completo.

Algunas partes dificiles, como el motor de sprites o una vista mas avanzada de la piscina, se pueden copiar y pegar primero. Despues, cuando ya tengas mas practica, las puedes estudiar con calma.

## Secciones

- `01_preparacion_macos`: preparar la computadora y conocer las ideas basicas.
- `02_flutter_basico`: aprender las piezas principales de Flutter.
- `03_swim_titans_paso_a_paso`: crear Swim Titans desde una app nueva.
- `04_retos`: ideas para seguir jugando y mejorando la app.

## Lo que haremos despues

Mas adelante cada leccion tendra explicaciones completas, pasos pequenos y ejemplos de codigo.

## Primera actividad

Empezar por `01_preparacion_macos/00_indice.md` y revisar que herramientas necesitas antes de crear la app.
