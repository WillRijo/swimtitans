# Crear proyecto

**Nivel:** Practicar

## Objetivo corto

Crear el proyecto nuevo donde construiremos Swim Titans desde cero.

## Archivos o herramientas que se usaran

- VS Code
- Flutter
- Terminal
- Carpeta nueva del proyecto

## Resumen de lo que se explicara despues

Se explicara como crear una app Flutter nueva y como elegir un nombre claro para el proyecto. Tambien se recordara que el proyecto de referencia no se modifica.
