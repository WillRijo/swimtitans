# Resumen del proyecto

Swim Titans es una app/juego educativo hecho con Flutter.

El juego muestra un nadador en una piscina. La persona que juega elige un tipo de nado, elige una distancia y practica con controles simples.

## Objetivo del juego

El objetivo del juego es llegar a la meta nadando la distancia seleccionada.

Las distancias son:

- 50 m
- 100 m
- 200 m
- 400 m

Cada distancia se divide en vueltas de 50 m.

## Objetivo educativo

El objetivo educativo es usar un juego simple para explicar conceptos basicos de programacion:

- Variables
- Estados
- Funciones
- Condiciones
- Enums
- Widgets
- Movimiento en pantalla
- Animacion con sprites

## Tecnologia usada

El proyecto usa:

- Flutter
- Dart

No usa motor de juegos externo.

## Que incluye la primera version

La primera version incluye:

- Pantalla de inicio
- Seleccion de modo practica
- Seleccion de tipo de nado
- Seleccion de distancia
- Pantalla de juego
- Piscina con tres carriles
- Movimiento por vueltas
- Sprites animados
- Reglas simples por tipo de nado

## Que no incluye todavia

Todavia no incluye:

- Backend
- Login
- Base de datos
- Lecciones completas
- Sistema de niveles
- Puntajes avanzados
- Modo competencia real
