# Crear primer proyecto

**Nivel:** Practicar

## Objetivo corto

Crear una app vacia de Flutter para ver que todo funciona.

## Archivos o herramientas que se usaran

- VS Code
- Flutter
- Terminal
- Un proyecto nuevo de prueba

## Resumen de lo que se explicara despues

Se explicara como crear el primer proyecto, abrirlo en VS Code y reconocer sus carpetas principales. Esta app sera solo para practicar antes de construir Swim Titans.

## Actividad futura breve

Crear el proyecto de prueba y ejecutarlo sin cambiar nada todavia.
