# Flutter basico

**Nivel:** Aprender y Practicar

## Objetivo

Conocer las piezas basicas de Flutter antes de construir Swim Titans.

## Herramientas que vamos a usar

- VS Code
- Flutter
- Dart
- Un proyecto Flutter de prueba
- Archivo `lib/main.dart`

## Lo que haremos en esta seccion

Primero vamos a mirar la app inicial de Flutter.

Luego aprenderemos ideas pequenas:

- donde empieza la app
- que es un widget
- como mostrar textos
- como acomodar cosas con Column y Row
- como usar botones
- que son variables
- que es estado
- como cambiar de pantalla

Todavia no construiremos todo Swim Titans aqui.

Esta seccion es para practicar piezas pequenas antes del juego.

## Primera actividad

Abrir `lib/main.dart` en una app de prueba y mirar que partes reconoces.
