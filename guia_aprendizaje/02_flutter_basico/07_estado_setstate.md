# Estado y setState

**Nivel:** Aprender

## Objetivo corto

Entender como una pantalla cambia cuando cambian sus datos.

## Archivos o herramientas que se usaran

- `StatefulWidget`
- `setState`
- Variables

## Resumen de lo que se explicara despues

Se explicara que el estado es la memoria de una pantalla. Cuando el estado cambia, Flutter puede volver a dibujar la pantalla usando `setState`.

## Actividad futura breve

Cambiar un contador simple y mirar como la pantalla se actualiza.
