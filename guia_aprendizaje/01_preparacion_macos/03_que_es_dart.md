# Que es Dart

**Nivel:** Aprender

## Objetivo

Entender que Dart es el lenguaje que usa Flutter para decirle a la app que hacer.

## Herramientas que vamos a usar

- Dart
- VS Code
- Archivos `.dart`

## Idea principal

Dart es un lenguaje de programacion.

Un lenguaje de programacion sirve para darle instrucciones a una computadora.

Con Dart podremos decirle cosas a Swim Titans como:

- muestra este texto
- cambia este color
- guarda esta distancia
- abre otra pantalla
- mueve el nadador
- revisa si el jugador llego a la meta

Los archivos que terminan en `.dart` son archivos donde escribimos codigo Dart.

Un ejemplo corto:

```dart
void main() {
  runApp(const AplicacionSwimTitans());
}
```

Ese codigo le dice a Flutter que empiece la app.

## Cosas que aprenderemos

Mas adelante usaremos Dart para aprender:

- variables
- funciones
- clases
- condiciones
- listas
- estados

No tienes que aprender todo eso ahora. Iremos paso a paso.

## Lo que haremos despues

Despues aprenderemos que significa programar y como dar instrucciones claras.

## Actividad

Escribe en una hoja tres instrucciones simples como si fueran ordenes para una app:

1. Muestra el titulo del juego.
2. Cambia el color del boton.
3. Mueve el nadador hacia adelante.
