# Instalar VS Code

**Nivel:** Practicar

## Objetivo

Instalar VS Code y entender que sera el lugar donde escribiremos el codigo.

## Herramientas que vamos a usar

- macOS
- Navegador
- VS Code

## Idea principal

VS Code es un editor de codigo.

Un editor de codigo es como un cuaderno especial para programar.

En VS Code podremos:

- abrir carpetas de proyectos
- crear archivos
- escribir codigo
- instalar extensiones
- usar la terminal integrada
- correr la app

## Paso 1: Descargar VS Code

1. Abre el navegador.
2. Busca `Visual Studio Code`.
3. Entra a la pagina oficial de VS Code.
4. Descarga la version para macOS.

## Paso 2: Instalar VS Code

1. Abre el archivo descargado.
2. Mueve **Visual Studio Code** a la carpeta **Applications** o **Aplicaciones**.
3. Abre VS Code desde **Applications** o buscandolo con Spotlight.

## Paso 3: Conocer las partes principales

- **Barra lateral:** muestra archivos y extensiones.
- **Editor:** es donde se escribe codigo.
- **Terminal integrada:** es donde se escriben comandos.
- **Extensiones:** agregan herramientas nuevas a VS Code.

## Paso 4: Abrir la terminal integrada

1. Abre VS Code.
2. Ve al menu **Terminal**.
3. Elige **New Terminal**.

Tambien puedes probar el atajo `Control + `` si funciona en tu teclado.

Esa terminal vive dentro de VS Code.

## Paso 5: Crear una nota de prueba

1. Crea un archivo temporal llamado `prueba.txt`.
2. Escribe: `Hola, estoy aprendiendo Flutter.`
3. Guarda el archivo.

Esto no es codigo todavia. Solo es practica para usar el editor.

## Si algo sale mal

- Si VS Code no abre, revisa que este en **Applications**.
- Si macOS pregunta si quieres abrir una app descargada de internet, confirma que si.
- Si no ves la terminal, busca el menu **Terminal** arriba.

## Lo que haremos despues

En la siguiente leccion instalaremos la extension de Flutter.

Tambien usaremos VS Code para instalar Flutter.

## Actividad

1. Descarga VS Code.
2. Abre VS Code.
3. Abre la terminal integrada.
4. Crea un archivo `prueba.txt`.
5. Escribe: `Hola, estoy aprendiendo Flutter.`
