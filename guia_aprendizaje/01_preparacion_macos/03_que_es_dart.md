# Que es Dart

**Nivel:** Aprender

## Objetivo corto

Entender que Dart es el lenguaje que usa Flutter para decirle a la app que hacer.

## Archivos o herramientas que se usaran

- Dart
- VS Code
- Archivos `.dart`

## Resumen de lo que se explicara despues

Se explicara que Dart es un lenguaje de programacion. Con Dart se pueden crear variables, funciones, clases y reglas para que la app responda cuando alguien toca un boton.
