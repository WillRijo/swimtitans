# Retos faciles

**Nivel:** Practicar

## Objetivo corto

Probar cambios pequenos para ganar confianza.

## Archivos o herramientas que se usaran

- Pantalla de inicio
- Pantalla de juego
- Textos
- Colores

## Resumen de lo que se explicara despues

Se propondran retos como cambiar colores, editar textos, agregar un boton nuevo o mostrar un mensaje cuando termina la carrera.

## Actividad futura breve
