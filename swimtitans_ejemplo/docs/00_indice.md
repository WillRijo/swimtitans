# Indice de documentacion tecnica

Esta carpeta contiene documentacion tecnica simple del proyecto Swim Titans.

El objetivo de estos documentos es ayudar a entender como esta organizada la app antes de crear las lecciones de aprendizaje.

La carpeta `guia_aprendizaje/` se creara despues. Esa guia sera para explicar programacion paso a paso.

## Documentos

1. `01_resumen_del_proyecto.md`
   - Explica que es Swim Titans y que incluye esta primera version.

2. `02_estructura_del_codigo.md`
   - Muestra el arbol principal de `lib/` y explica para que sirve cada archivo.

3. `03_flujo_de_la_app.md`
   - Explica como la app pasa desde `main.dart` hasta la pantalla de juego.

4. `04_modelos.md`
   - Explica los enums y los datos principales del juego.

5. `05_pantalla_inicio.md`
   - Explica la pantalla donde se elige tipo de nado y distancia.

6. `06_pantalla_juego.md`
   - Explica la pantalla donde ocurre la practica.

7. `07_sprites_y_assets.md`
   - Explica las imagenes, sprites y animaciones.

8. `08_reglas_del_juego.md`
   - Explica los controles y las distancias.

9. `09_terminal_y_comandos.md`
   - Explica comandos utiles para trabajar con el proyecto.

10. `10_pubspec_y_assets.md`
    - Explica el archivo `pubspec.yaml`, assets e iconos.
