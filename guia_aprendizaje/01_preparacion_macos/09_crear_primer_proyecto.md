# Crear primer proyecto

**Nivel:** Practicar

## Objetivo

Crear una app vacia de Flutter para confirmar que todo funciona.

## Herramientas que vamos a usar

- VS Code
- Flutter
- Command Palette
- Terminal integrada
- Navegador o simulador

## Idea principal

Antes de crear Swim Titans vamos a crear una app de prueba.

Esta app sirve para confirmar que Flutter esta instalado correctamente.

No hay que modificar esa app todavia.

## Paso 1: Abrir VS Code

1. Abre VS Code.
2. Cierra proyectos anteriores si hace falta.

## Paso 2: Abrir Command Palette

1. Presiona `Command + Shift + P`.
2. Escribe `Flutter`.
3. Elige **Flutter: New Project**.

## Paso 3: Elegir tipo de proyecto

Elige **Application**.

Esto crea una app normal de Flutter.

## Paso 4: Elegir carpeta

Elige una carpeta facil de encontrar.

Por ejemplo:

```text
~/developer
```

Ahi se guardara el proyecto.

## Paso 5: Nombrar el proyecto

Usa este nombre simple:

```text
mi_primera_app
```

Para nombres de proyectos:

- usa letras minusculas
- usa guion bajo si hay varias palabras
- no uses espacios

## Paso 6: Esperar que VS Code cree el proyecto

VS Code va a crear carpetas y archivos.

Puede tardar un poco.

Cuando termine, se abrira el proyecto.

## Paso 7: Ejecutar la app

Puedes elegir una de estas opciones.

**Opcion A**

Presiona `F5`.

Elige Chrome o un simulador si VS Code pregunta.

**Opcion B**

Abre la terminal integrada y ejecuta:

```bash
flutter run
```

La app inicial normalmente muestra un contador.

Si ves la app, Flutter esta funcionando.

## Paso 8: Detener la app

Si se ejecuto con `F5`, usa el boton **Stop**.

Si se ejecuto en terminal, presiona `q` si aparece disponible o `Control + C`.

## Si algo sale mal

- Si no aparece ningun dispositivo, prueba con Chrome.
- Si `flutter run` muestra error, copia el mensaje.
- Si VS Code no encuentra Flutter, vuelve a revisar Flutter Doctor.
- No borres el proyecto sin entender el error.

## Lo que haremos despues

Despues aprenderemos las partes basicas de la app inicial.

Luego crearemos Swim Titans paso a paso.

## Actividad

1. Crea un proyecto llamado `mi_primera_app`.
2. Ejecutalo sin cambiar nada.
3. Escribe que viste en la pantalla.
4. Cierra la app cuando termines.
