# Botones

**Nivel:** Practicar

## Objetivo corto

Aprender a poner botones y hacer que respondan al tocarlos.

## Archivos o herramientas que se usaran

- `lib/main.dart`
- `ElevatedButton`
- `onPressed`

## Resumen de lo que se explicara despues

Se explicara que un boton puede ejecutar una accion cuando se toca. Tambien se vera como escribir una accion pequena dentro de `onPressed`.
