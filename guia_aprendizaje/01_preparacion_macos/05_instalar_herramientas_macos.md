# Instalar herramientas en macOS

**Nivel:** Practicar

## Objetivo corto

Preparar macOS con las herramientas necesarias para trabajar con Flutter.

## Archivos o herramientas que se usaran

- macOS
- Navegador
- Terminal
- Git
- Xcode Command Line Tools

## Resumen de lo que se explicara despues

Se explicara que herramientas necesita Flutter en macOS y como revisar que la computadora esta lista. La leccion sera paso a paso y sin ir rapido.

## Actividad futura breve

Abrir Terminal y aprender a escribir un comando simple sin miedo.
