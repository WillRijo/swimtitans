# Crear piscina simple

**Nivel:** Practicar

## Objetivo corto

Dibujar una piscina sencilla en la pantalla del juego.

## Archivos o herramientas que se usaran

- Pantalla de juego
- `Container`
- `Stack`
- Colores

## Resumen de lo que se explicara despues

Se explicara como usar cajas y colores para representar una piscina simple con carriles. Esta sera una primera version, antes de usar graficos mas avanzados.
