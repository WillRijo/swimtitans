# Swim Titans

Un juego educativo hecho con Flutter para aprender programacion paso a paso.

> 🕊️ Lucas!, Lhuna te dedica esta guía de aprendizaje. 🕊️

[🎮 Jugar demo](https://willrijo.github.io/swimtitans/demo/)

## Que es este proyecto

Swim Titans es una app/juego simple de natacion creada para aprender Flutter, Dart y conceptos basicos de programacion.

## Demo

https://willrijo.github.io/swimtitans/demo/

## Estructura del repositorio

- `swimtitans_ejemplo/`: proyecto Flutter de referencia.
- `demo/`: version web publicada.
- `guia_aprendizaje/`: guia paso a paso para aprender desde cero.

## Objetivo educativo

La idea es aprender:

- que es Flutter
- que es Dart
- widgets
- variables
- estado
- navegacion
- sprites
- reglas simples de juego

## Tecnologias

- Flutter
- Dart
- GitHub Pages
